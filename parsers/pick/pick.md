# PICK

## Description

Loop on all tokens and get only keys given in params.

## Interface 
```ts
interface x {
  "name": "pick",
  "options"?: {
    "keys": Array<string>
  }
}
```

## Options example
```json
{
  "name": "pick",
  "options": {
    "keys": ["name", "type"]
  }
}
```

## Types

### Input

Array of object with at least a name's key

```ts
Array<{[key: string]: any}>
```

### Output

Array of object with the keys given in options

```ts
Array<{[key: string]: any}>
```
